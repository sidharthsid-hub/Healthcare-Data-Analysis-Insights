# Healthcare Data Analysis and Insights

## 📋 Project Overview
This project focuses on analyzing a multi-dimensional healthcare dataset comprising medical examinations, hospitalization details, and customer profiles. The goal was to extract actionable insights into patient health profiles, identify correlations between medical history and healthcare costs, and provide a visual tool for stakeholders to make informed decisions.

## 🎯 Objectives
- **Data Cleaning:** Handled missing values (marked as '?'), standardized date formats, and imputed missing categorical/numerical data.
- **Data Transformation:** Merged multiple datasets (Medical, Hospitalization, and Customer details) into a master database using VLOOKUP and Index-Match.
- **Descriptive Analysis:** Explored the relationships between health metrics like BMI, HbA1C, and costs.
- **Data Visualization:** Built interactive charts and a comprehensive dashboard to simplify complex health trends.

## 🛠️ Tools Used
- **Microsoft Excel:** Data Cleaning, Pivot Tables, Advanced Formulas (VLOOKUP, IF, Statistical functions).
- **Excel Charts:** Pie/Donut charts, Bar/Column charts, and Scatter plots.
- **Slicers:** For building an interactive user experience.

## 🔍 Key Insights & Analysis
- **Cancer & Smoking:** Visualized the distribution of cancer history across smokers vs. non-smokers.
- **Transplant Impact:** Compared how transplant history affects the average number of major surgeries and HbA1C levels.
- **Cost Analysis:** - Identified how healthcare charges vary significantly based on **Weight Status** and **Diabetes Status**.
  - Compared average charges across different **Hospital Tiers** and **States**.
- **Correlations:** Analyzed the relationship between **Age** and healthcare costs, as well as Age vs. BMI/HbA1C using scatter plots.

## 📊 Dashboard Features
The final dashboard consolidates all key metrics into a single view:
- **Interactive Slicers:** Users can filter the entire dashboard by *Weight Status* and *Diabetes Status*.
- **Visual Clarity:** Includes a mix of distribution charts, comparative bar charts, and trend-focused scatter plots.
